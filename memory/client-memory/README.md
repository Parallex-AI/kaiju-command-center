# Client Memory

Stores persistent context about each client to improve agent responses over time.

## Purpose

Agents use client memory to:
- Understand historical campaign performance
- Recall past recommendations and their outcomes
- Personalize analysis and tone per client

## Structure (planned)

Each client has its own memory file or record containing:
- Client name and account identifiers
- Historical metrics snapshots
- Past recommendations
- Noted preferences and constraints
